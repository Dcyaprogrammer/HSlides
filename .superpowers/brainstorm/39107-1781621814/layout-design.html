Navigation & Interactions

  • Hover states: Cards lift up with shadow offset (translate + box-shadow)
  • Colors: Each card gets a different brutalist accent color (pink, cyan, lime, yellow, violet)
  • Typography: Bold, compressed fonts for headings. System fonts for body.
  • Responsiveness: Grid collapses to 2 columns on tablet, 1 on mobile
