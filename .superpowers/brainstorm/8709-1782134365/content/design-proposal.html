HSlides UI/UX Improvements - Aurora Tech Design

Elevating neo-brutalism with aurora gradients and refined modern aesthetics

1. Color Palette Refinement

问题：7种扁平颜色（cyan, pink, lime, violet, yellow, orange, red）过于杂乱

解决方案：精简为 3 种主渐变色调

Primary Gradients:

渐变 A: 紫蓝紫 | 渐变 B: 粉红橙 | 渐变 C: 青蓝

应用方式：

  • 卡片背景：白色底色 + 渐变叠加层（30% 不透明度）
  • 按钮：纯渐变背景 + 黑色边框
  • 悬停：渐变叠加层 opacity 30% → 50%

2. Border & Shadow Refinement

Before (Current)

3px 边框 + 硬阴影

After (Proposed)

2px 边框 + 柔和模糊阴影

Changes:

  • Border: `border-3px` → `border-[2px]`
  • Shadow: `box-shadow: 4px 4px 0px #000` → `box-shadow: 0 4px 20px rgba(0,0,0,0.15)`
  • Result: 更精致、更现代的视觉效果

3. Typography Upgrade

Current: System fonts (Inter, -apple-system)
HSlides - Discover beautiful HTML slides
Proposed: Geist Variable (Vercel's choice)
HSlides - Discover beautiful HTML slides

Benefits: 更现代、更专业、更好的可变字重支持

4. Spacing Improvements

Cards (current): p-4 (16px padding)

Cards (proposed): p-5 (20px padding) - 适度增加

Grid gaps: gap-5 (20px) → gap-6 (24px)

Hero section: py-12 (48px) → py-16 (64px)

Result: 保持紧凑感的同时，视觉更舒适呼吸

5. Animation Refinements

Card hover (current): transition-all duration-200, -translate-x-1 -translate-y-1

Card hover (proposed):
-translate-x-1.5 -translate-y-1.5
transition-all duration-300
+ scale-105 (轻微缩放)

Modal (current): fade-in 0.3s, scale-in 0.3s

Modal (proposed): fade-in 0.4s, scale-in 0.4s (更流畅)

Result: 200-300ms 响应，明显的位移和阴影变化

Summary of Changes

  • ✅ Colors: 7 colors → 3 gradients
  • ✅ Borders: 3px → 2px, softer
  • ✅ Shadows: Hard → Soft blurred shadows
  • ✅ Typography: System fonts → Geist Variable
  • ✅ Spacing: Moderate increase (4px → 5px, 5px → 6px)
  • ✅ Animations: 200-300ms, scale effect, smooth transitions

Aesthetic Goal:

Linear 风格的科技未来感 - 柔和渐变 + 玻璃态 + 现代字体 + 精致细节
